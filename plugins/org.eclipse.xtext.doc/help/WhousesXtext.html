Who uses Xtext?

Xtext is used in many different industries. It is used in the field of mobile devices, automotive development, embedded or Java enterprise software projects and game development. People use Xtext-based languages to generateJava, C, C++, C#, Objective C, Python, or Ruby code. Although the infrastructure runs on the JVM, you can compile Xtext languages to any existing target platform. Xtext based languages are developed for well known Open-Source projects such as Maven, Eclipse B3, the Eclipse Webtools platform or Google’s Protocol Buffers and the framework is also widely used in research projects.
