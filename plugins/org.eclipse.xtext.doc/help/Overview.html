What is Xtext?

No matter if you want to create a small textual domain-specific language (DSL) or you want to implement a full-blown general purpose programming language. With Xtext you can create textual programming languages in a snap. Also if you already have an existing language but it lacks decent tool support, you can use Xtext to create an Eclipse-based IDE providing editing experience known from modern Java IDEs in a surprisingly short amount of time. We call Xtext a language development framework.
